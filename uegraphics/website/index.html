Universe Exploration

Good old clunky space strategy... or something else?

Universe Exploration is a strategy game where one must travel through star systems in search for civilized life. To keep on searching, you must not only look for resources but also take care of your crew morale.

Universe Exploration is still a work-in-progress where I am experimenting with libGDX library written in Java. The source code is open for fork, reviews and so forth. There are many lacking features.

Source code

Project is available on GitHub: https://github.com/temedeus/universe-exploration

Feel free to fork and gaze around the source code.
